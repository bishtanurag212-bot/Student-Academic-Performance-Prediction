## 🎯 How to Use

### 1. Enter Student Data
Fill in the student information form with:
- Academic metrics (attendance, scores, study hours)
- Lifestyle factors (sleep, extracurricular)
- Support indicators (teacher support, resources)
- Additional factors (financial stress, career confidence)

### 2. Generate Prediction
Click "Predict Performance" to get:
- Performance classification
- Confidence score
- Risk assessment
- Personalized recommendations

### 3. Analyze Results
Review the comprehensive analysis:
- Performance category
- Risk score
- Probability distribution
- Key influencing factors
- Actionable recommendations

## 🚀 Features

- ✅ **Multi-Dataset Support**: University, High School, Online, and International students
- ✅ **Advanced ML Models**: Random Forest, XGBoost, LightGBM, SVM, Logistic Regression
- ✅ **Real-time Predictions**: Predict student performance with confidence scores
- ✅ **Risk Assessment**: Identify at-risk students with comprehensive risk scores
- ✅ **Personalized Recommendations**: AI-generated intervention strategies
- ✅ **Interactive Dashboard**: Visual analytics and trend analysis
- ✅ **Multi-Dataset Insights**: Compare performance across different institutions

---

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **ML Libraries**: scikit-learn, XGBoost, LightGBM
- **Visualization**: Plotly, Matplotlib, Seaborn
- **Data Processing**: Pandas, NumPy
